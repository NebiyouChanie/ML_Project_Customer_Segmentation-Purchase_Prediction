# Customer Purchase Prediction App

This Streamlit application demonstrates the deployment of a machine learning
model that predicts whether a customer is likely to make a purchase.

## Features
- Interactive customer input
- Pre-trained ML model inference
- Probability-based prediction output
- Simple and clean UI

## Tech Stack
- Python
- Streamlit
- Scikit-learn

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
